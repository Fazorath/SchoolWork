Form Requirements:

  • Submit and reset buttons
  • Checkboxes
  • Radio buttons
  • Drop-down list
  • Labels where appropriate
  • Fieldset and legend
  • Textarea element
  • Placeholder
  • Required attribute
  • Validation (email, URL, or tel)
  • Color, number/range, or date control

Table Requirements:

  • Table with thead, tbody, and tfoot elements
  • Minimum of 5 rows and 2 columns
  • Colspan or rowspan attribute
  • Caption

Audio/Video Requirements:

  • Audio element with audio file
  • Video element with video file
  • Embed element with YouTube video

CSS3 Techniques:

  • Accordion
  • Transition
  • Transform
  • Animation
  • Filter
